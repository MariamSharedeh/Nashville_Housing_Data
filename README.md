SQL Data Cleaning Project – Nashville Housing Dataset
 Project Overview

This project demonstrates data cleaning in SQL Server using a real dataset called “Nashville Housing”, which contains information about property sales in Nashville, Tennessee (USA).
The goal of this project is to transform messy raw data into clean, standardized, and analysis-ready data.

 Objectives

✅ Standardize date formats

✅ Fill in missing property addresses

✅ Split addresses into separate columns (Address, City, State)

✅ Normalize categorical values (Yes/No)

✅ Detect and remove duplicates

✅ Drop unnecessary columns

 Technologies Used

Microsoft SQL Server

T-SQL (Transact-SQL)

Window Functions (ROW_NUMBER, PARTITION BY)

String Functions (SUBSTRING, CHARINDEX, PARSENAME)

Conditional Functions (CASE WHEN, COALESCE, ISNULL)
